What is JQL?

JQL (Jexplorer Query Language) is the structured query language used when doing queries (aka searches) inside Jexplorers query bar


The query bar.

Typing in JQL into the query bar can be used to only show specific Instances within the explorer, only with the conditions you specified. For example, typing in Name = Part & ClassName = Folder will only show all Instances who have properties where Name == "Part" and ClassName == "Folder"

Syntaxes

Alias Usage Syntax Example
Equality Condition only passes if instance contains the property with the specified value. Value is written in JDF without quotes NAME = VALUE Name = Part
And Condition only passes if both the left and right sides pass. [LEFT] & [RIGHT] Name = Part & ClassName = Part
And Condition only passes if either the left or right side passes. [LEFT] | [RIGHT] Name = Part | ClassName = Part
Parent Modifies a condition to apply to the parent of an instance instead of itself. This operator is stackable <[CONDITION] <<Name = Workspace & <<<ClassName = DataModel
First Child Modifies a condition to apply to the first children of an instance that matches the condition. This operator is stackable >[CONDITION] >>Name = Light
