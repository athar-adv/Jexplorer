# Getting Started

## NOTICE: Documentation is currently a stub. It will be developed more

To get started, download the Jexplorer plugin either via Roblox's Marketplace or from the latest release.

[Marketplace](https://create.roblox.com/store/asset/108870283294163/Jexplorer?viewFromStudio=true&keyword=&searchId=7930a7dc-d0ec-4706-b1e7-8f77d9982c33)
[GitHub Releases](https://github.com/athar-adv/Jexplorer/releases)

If you're downloading from the Marketplace, click the `Plugins` dropdown in the top bar and `Manage Plugins` and turn on the Jexplorer plugin in Roblox Studio.

If you're downloading from GitHub, click the `Plugins` dropdown in the top bar and `Plugins Folder` and paste in your Jexplorer.rbxm in there.

Once you've installed the plugin, head over to the `Plugins` tab in the Roblox Studio ribbon and you should see a button with the Jexplorer icon. Click that and the plugin widget will open and you can start using it!
